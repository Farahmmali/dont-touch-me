
# 🗑️ Don't Touch Me
### Smart Touchless Trash Bin

**Throw waste. Not germs.**

A smart trash bin that opens automatically when you move your hand near it.
No touching. No germs. Just clean and simple tech.

---

# 🚀 Project Idea

Ever touched a trash bin lid and instantly regretted it?
Yeah… we thought so too.

So we built a **smart trash bin** that opens for you using sensors and an Arduino.
Just move your hand near the lid and it opens automatically.

Clean. Easy. Smart.

---

# ⚙️ How It Works

1️⃣ A **distance sensor** detects your hand.  
2️⃣ The sensor sends a signal to the **Arduino**.  
3️⃣ The Arduino activates a **servo motor**.  
4️⃣ The lid opens automatically.  
5️⃣ After a few seconds, it closes again.

All of this happens in seconds without touching anything.
